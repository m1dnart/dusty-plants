# Інструкція

1. На сайті <https://jeanropke.github.io/RDOMap/> треба увімкнути `Show Debug Settings` в розділі `Settings`. Нижче появиться розділ `Debug` і там треба увімкнути `Enable debug features`.
2. Після цього на відмітках вже існуючих рослин (треба увімкнути всі) знизу буде відображатись наступне - `Latitude: -88.9783 / Longitude: 86.1932`. Копіюємо цей текст.
3. Запускаємо `script.exe`. Вставляємо текст в інпут.
4. Потім на карті, де `User Pins` треба через `Browse` вказати шлях до файлу `dusty_plants.txt` та натиснути `Import`. В розділі `Utilities` мають треба увімкнути `User Pins`. Після кожного наступного додавання достатньо буде просто натиснути `Import` і мапа оновиться. Кастомні маркери перекривають всі інші. Таким чином можна бачити чітко, що було відмічено, а що ні.

**Примітка**: Щоб просто використовувати маршрути, то достатньо завантажити `dusty_plants.txt` (дивись п.4). Для зручності всі інші маркери варто приховати.
