# Інструкція (неактуально)

1. На сайті <https://jeanropke.github.io/RDOMap/> треба увімкнути `Show Debug Settings` в розділі `Settings`.
2. Після цього на відмітках вже існуючих рослин (треба увімкнути всі) знизу буде відображатись наступне - `Latitude: -88.9783 / Longitude: 86.1932`. Копіюємо цей текст.
3. Запускаємо `dusty-plants-script.exe`. Вставляємо текст в перший інпут. В другому вибираємо рослину, вказавши її номер.
4. Потім на карті, де `User Pins` треба через `Browse` вказати шлях до файлу `dusty_plants.txt` та натиснути `Import`. Після кожного наступного додавання достатньо буде просто натиснути `Import` і мапа оновиться. Кастомні маркери перекривають всі інші. Таким чином можна бачити чітко, що було відмічено, а що ні.
5. **Це якщо хочеться додавати**, якщо просто використовувати, то достатньо завантажити `dusty_plants.txt` і в розділі `Utilities` вибрати `User Pins`. Для зручності всі інші маркери варто приховати.
6. DONE.

**Примітка:** бажано додавати рослини окремо, щоб не було дублікатів. Після завершення кидайте файли `dusty_plants.json` та `dusty_plants.txt` мені.
